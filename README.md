# Fotress Drone Solutions

Modern, premium marketing site for **Fotress Drone Solutions** (drone sales + repair + technical support).

## Routes

- `/` Home
- `/about` About Us
- `/log-queries` Log Queries
- `/track-repair` Track Drone Repair
- `/admin/sign-in` Admin Sign In (frontend-only demo)

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
